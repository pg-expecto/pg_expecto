## История версий

- 03.01.2026 : **версия 5.0** : Сбор статистических данных по обработке блоков разделяемой памяти, формирование входных файлов нейросети для подготовки отчетов по нагрузочному тестирования.
- 10.11.2025 : В рамках **версии 4.0** реализована поддержка гибкой настройки тестовых сценариев, а также предусмотрено применение обособленной базы данных в методологии нагрузочного тестирования.
- 05.11.2025 : Версия **3.1** - Добавлена возможность изменения сценариев нагрузочного тестирования
- 01.11.2025 : Доступная версия продукта **3.0**
