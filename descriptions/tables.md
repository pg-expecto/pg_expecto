## SQL файлы

- `core_cluster_tables.sql`
- `core_os_tables.sql`
- `core_statement_tables.sql`
- `load_test_tables.sql`
- `markov_chain_tables.sql`

## Таблицы

| Наименование файла | Наименование таблицы | Краткое описание таблицы |
|--------------------|----------------------|---------------------------|
| `core_cluster_tables.sql` | `cluster_stat` | Исходные данные для расчётов метрик производительности и ожиданий (текущая статистика кластера). |
| | `cluster_stat_median` | Скользящие медианы метрик производительности и ожиданий. |
| | `performance_incident` | Логирование инцидентов производительности (время начала и окончания). |
| `core_os_tables.sql` | `os_stat_vmstat` | Метрики vmstat (процессы, память, swap, IO, прерывания, CPU, dirty страницы и т.д.). |
| | `os_stat_vmstat_median` | Скользящие медианы по метрикам vmstat. |
| | `os_stat_iostat_device` | Метрики iostat для дисковых устройств (операции чтения/записи, загрузка, очереди). |
| | `os_stat_iostat_device_median` | Скользящие медианы по метрикам iostat. |
| `core_statement_tables.sql` | `statement_stat` | Исходные данные производительности и ожиданий SQL-запроса (вызовы, строки, wait_stats в JSONB). |
| | `statement_stat_sql` | Текст SQL-выражения по queryid. |
| | `statement_stat_wait_events` | Детализированные данные ожиданий на уровне SQL (тип и событие ожидания). |
| | `statement_stat_median` | Скользящие медианы метрик на уровне SQL. |
| | `statement_stat_waitings_median` | Скользящие медианы по событиям ожидания для SQL-выполнения. |
| `load_test_tables.sql` | `load_test` | Параметры нагрузочного теста (подключения, тайминги, настройки ядра ОС). |
| | `load_test_pass` | Итерации (проходы) нагрузочного теста с временными метками и текущей нагрузкой. |
| | `testing_scenarios` | Связь тестовых сценариев с SQL-запросами и весами. |
| `markov_chain_tables.sql` | `markov_config` | Конфигурация цепи Маркова (скорость забывания, интервалы, пороги, флаги адаптивности). |
| | `mchain_error_log` | Журнал ошибок, возникающих при работе функций цепи Маркова. |
| | `markov_frequencies` | Накопленные частоты переходов между состояниями (основа для расчёта вероятностей). |
| | `transition_log` | Посекундный/минутный журнал всех переходов между состояниями. |
| | `markov_probabilities` | Матрица вероятностей переходов, рассчитанная из частот. |
| | `markov_absorbing` | Поглощающая матрица для многошагового прогноза риска (аварийные состояния — поглощающие). |
| | `state_descriptions` | Справочник всех 189 состояний (корреляция, тренды OS и ожиданий). |
| | `markov_chain` | Текущее и предыдущее состояние цепи (одна строка для пошагового обучения). |
| | `apply_forgetting_log` | Лог всех вызовов функции забывания (применённый alpha, детали расчёта). |
