# Дополнительные возможности по настройкам нагрузочного тестирования

## Постоянная нагрузка 
**Отрицательный вес сценария.**
```
scenario1 = -3
```
**Результат** : Постоянное количество соединений = 3, по сценарию-1 

## Бесконечный тест с нагрузкой по распределению Пуассона

```
# БЕСКОНЕЧНЫЙ ТЕСТ. 
# ДЛЯ ОСТАНОВКИ 
# /postgres/pg_expecto/sh/load_test/load_test_stop.sh
period_hours = -1
average_load = 40
```
**Результат** : Тест не будет остановлен , средняя количество сессий в каждой итерации теста = 40
